# Conéctate con la API

LAB *Obligatorio
1. Conéctate con la API "https://jsonplaceholder.typicode.com/posts"

2. Haz un CRUD completo (Leer los datos, crear datos nuevos, actualizar datos y borrar datos).

La lectura debe hacerse en el momento en el que carga el componente. 

La creación debe realizarse cuando se le da al botón "crear".

La actualización debe hacerse cuando se le da al botón "actualizar".

El borrado deberá hacerse, tras realizar una pregunta de seguridad (usa "confirm") y tras haber hecho clic en el botón "borrar".

Puedes realizar este ejercicio usando JavaScript Vanilla o cualquier framework entre Angular, React o VUE.
