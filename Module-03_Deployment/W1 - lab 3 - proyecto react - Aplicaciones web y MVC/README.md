# Proyecto REACT

LAB *Obligatorio

Crea tu propio proyecto REACT que contenga: 

- Un archivo JSON con canciones que te gusten (5 canciones)

Cada canción debe tener: 

{
titulo: String,
album: String,
imagenAlbum: String (url)
duracion: integer,
valoracion: integer
}

- Un componente que sirva para mostrar la lista de canciones

- Un componente que muestre información sobre tus aficiones

Recuerda que no es obligatorio aplicar CSS, pero que te lo recomiendo encarecidamente.
