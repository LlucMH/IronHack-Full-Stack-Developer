# Proyecto React

LAB *Obligatorio

En el lab de hoy deberás crear tu primer proyecto de React. 

Deberás añadir: 

- h1 Con tu nombre
- h2 Con tus apellidos
- Lista de aficiones
- Una imagen
- Un enlace a cualquier página de interés (como google)

Tras esto, sube el proyecto a un repositorio y entrégalo.
